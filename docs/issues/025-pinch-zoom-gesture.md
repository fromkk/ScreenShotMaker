# Issue #025: ピンチジェスチャーによるズーム

## Phase / Priority
Phase 5 | P2 (Medium)

## 概要

トラックパッドの2本指ピンチイン・ピンチアウトジェスチャーで、キャンバスのズームをなめらかに拡大・縮小できるようにする。現在はボタン (+ / -) でのみズーム操作が可能。

## 現状の問題

1. ズーム操作が + / - ボタンのみ (0.1 刻み)
2. トラックパッドジェスチャーに対応していない
3. ジェスチャーによる直感的な操作ができない

## 対象ファイル

- 変更: `ScreenShotMaker/Views/CanvasView.swift` (ジェスチャー追加)

## 実装詳細

1. **MagnifyGesture の追加**
   - CanvasView の ScrollView またはキャンバスコンテンツに `.gesture(MagnifyGesture())` を追加
   - `@GestureState` でジェスチャー中の一時的なスケール変更を管理
   - ジェスチャー終了時に `zoomScale` を更新

2. **スムーズなズーム**
   - ジェスチャー中はリアルタイムでプレビュー倍率を更新
   - `zoomScale` の範囲制限 (0.1〜3.0) を適用
   - アニメーション付きで滑らかに変化

3. **既存ボタンとの共存**
   - + / - ボタンによるズームは引き続き動作
   - ジェスチャーとボタンで同じ `zoomScale` を共有

## 受け入れ基準

- [ ] トラックパッドのピンチインで縮小される
- [ ] トラックパッドのピンチアウトで拡大される
- [ ] ジェスチャー中のズームがスムーズ（カクつかない）
- [ ] ズーム範囲の上限・下限で止まる
- [ ] 下部のズーム %表示がリアルタイムで更新される
- [ ] + / - ボタンとジェスチャーが競合しない
