# Issue #026: キーボードショートカットによるズーム + 100% リセット

## Phase / Priority
Phase 5 | P2 (Medium)

## 概要

キーボードショートカットでズーム操作を行えるようにする。⌘+ / ⌘- で10%刻みのズーム、⌘0 で100%表示にリセット。また、画面下部のズーム %表示をクリックして100%にリセットできるようにする。

## 現状の問題

1. キーボードショートカットでズーム操作ができない
2. 100%表示にリセットする手段がない
3. 下部の %表示が読み取り専用でインタラクション不可

## 対象ファイル

- 変更: `ScreenShotMaker/Views/CanvasView.swift` (ショートカット・リセット機能)
- 変更: `ScreenShotMaker/Views/ContentView.swift` (メニューバーコマンド)

## 実装詳細

1. **キーボードショートカットの追加**
   - ⌘+ (Command + Plus/Equal): ズームイン 10%刻み
   - ⌘- (Command + Minus): ズームアウト 10%刻み
   - ⌘0 (Command + Zero): 100%表示にリセット
   - `.keyboardShortcut()` modifier または `.commands {}` で実装

2. **メニューバー統合**
   - View メニューに「Zoom In」「Zoom Out」「Actual Size」を追加
   - 各メニュー項目にショートカットキー表示

3. **100% リセット機能**
   - ⌘0 で `zoomScale = 1.0` に設定
   - 下部の %表示テキストを `Button` に変更し、クリックで100%にリセット
   - リセット時にアニメーション付きでスムーズに移行

4. **zoomScale の状態共有**
   - CanvasView の `zoomScale` を親ビュー (ContentView) に引き上げるか、
   - `@Binding` で共有してメニューコマンドからアクセス可能にする

## 受け入れ基準

- [ ] ⌘+ でズームインされる (10%刻み)
- [ ] ⌘- でズームアウトされる (10%刻み)
- [ ] ⌘0 で100%表示にリセットされる
- [ ] 下部の %表示クリックで100%にリセットされる
- [ ] View メニューにズーム操作が表示される
- [ ] ズーム範囲の上限・下限を超えない
- [ ] リセット時にスムーズなアニメーションが適用される
